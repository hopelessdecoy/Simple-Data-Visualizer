Welcome to Data Visualizer!

1. First row is column names
2. All Data must be delimited by ":"
3. All fields but the first must be integer values 
(ex: 5, number of clicks, any integer, decimals will be rounded!)
4. The first column is used as a key (ex: company name, URL, ect)
5. File must be named "data.txt" and be placed in the data folder
6. Start the exe and go!

NOTE: Doesn't register negative numbers yet :(

NOTE 2: Use Arrow keys to scroll!
